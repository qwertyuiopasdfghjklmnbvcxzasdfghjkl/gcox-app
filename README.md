# x-game

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Prepare app
```
cordova prepare
```

### Build Android app
```
cordova build android
```


### Build iOS app ( macOS required)
```
cordova build ios
```





### Run your tests
```
npm run test
```

### Lints and fixes files
```
npm run lint
```
